💡 Не реагируют каналы с CH9 по CH16?

По умолчанию пульт передает базовый пакет каналов. Чтобы активировать остальные:

  1. На самом пульте зайдите в меню текущей модели (кнопка MDL -> SETUP).
  2. Проверьте параметр 'Internal RF / External RF' — диапазон каналов должен быть выставлен как 'CH1-CH16'.
  3. Перейдите на вкладку 'MIXERS' (Микшеры) на пульте и вручную назначьте нужные физические тумблеры (например, SA, SB) на свободные каналы с 9 по 16, иначе компьютер их не увидит.
